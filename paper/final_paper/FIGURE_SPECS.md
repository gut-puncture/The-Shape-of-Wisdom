# Figure Specifications - Shape of Wisdom

All paper figures are generated by `sow.v2.figures.paper_figures` using stored parquet outputs only.
No model inference is run during figure generation.

## Regenerate (CPU only)

```bash
python scripts/v2/regenerate_paper_figures.py \
  --parquet-dir results/parquet \
  --output-dir paper/final_paper/figures
```

## Regenerate + Verify + PNG previews

```bash
python scripts/v2/regenerate_paper_figures.py \
  --parquet-dir results/parquet \
  --output-dir paper/final_paper/figures \
  --verify
```

This verification mode renders PNG previews in `tmp/fig_previews/`, runs the in-code layout integrity gate (overlap + out-of-canvas + annotation checks), and prints per-figure status.

## Figure 1 - Three Primitives (`fig1_primitives.pdf`)

- Layout: 1x3 full-width cards.
- Panels:
  - (a) STATE: mean `delta(l)` by trajectory type.
  - (b) MOTION: mean `drift g(l)` by trajectory type.
  - (c) BOUNDARY: mean `|delta(l)|` by trajectory type with `b_min=0.3` overlay.
- Styling logic:
  - Stable trajectories: thicker solid lines.
  - Unstable trajectories: lighter dashed lines.
  - Faint ghost trajectories for stable types show per-prompt variability.
  - Boundary panel shades region below `b_min`.
- Data files: `decision_metrics.parquet`, `prompt_types.parquet`.

## Figure 2 - Phase Diagram (`fig2_phase_diagram.pdf`)

- Layout: one-column scatter + right marginal histogram.
- Axes:
  - X: `|delta(L-tau)|` at tail entry, `tau=8`.
  - Y: tail sign-flip count in the last 8 layers.
- Visual structure:
  - Threshold lines at `b_min=0.3` and flip threshold `0.5`.
  - Light region shading for stable/unstable interpretation.
  - Covariance ellipse per trajectory type.
  - Discrete-friendly marginal histogram of flip counts.
- Data files: `decision_metrics.parquet`, `prompt_types.parquet`.

## Figure 3 - Drift Decomposition (`fig3_decomposition.pdf`)

- Layout: headline row (2 panels) + diagnostics row (3 panels).
- Panels:
  - (a) Stable-correct traces: mean `s_attn`, `s_mlp`, observed `g`, with SEM bands.
  - (b) Stable-wrong traces: same metrics.
  - (c) Early vs late positive-steering share (attention vs MLP) with bootstrap 95% CI.
  - (d) Sign-alignment fraction `sign(s_attn)==sign(s_mlp)` by trajectory type with bootstrap CI.
  - (e) Oscillation proxy (drift sign-change rate) by trajectory type with bootstrap CI.
- Data files: `tracing_scalars.parquet`, `decision_metrics.parquet`, `prompt_types.parquet`.

## Figure 4 - Attention Routing (`fig4_attention_routing.pdf`)

- Layout: left stacked-mass small multiples + right contribution heatmap.
- Panels:
  - (a) Stable-correct role-normalized attention-mass stack by depth.
  - (b) Stable-wrong role-normalized attention-mass stack by depth.
  - (c) Heatmap of mean decision-aligned attention contribution by span role x trajectory type.
- Span-role order:
  - `instruction`, `question_stem`, `option_correct`, `option_competitor`, `option_other`, `post_options`.
- Heatmap values are in logit-margin units; diverging scale is centered at zero.
- Data files: `attention_mass_by_span.parquet`, `attention_contrib_by_span.parquet`, `decision_metrics.parquet`, `prompt_types.parquet`.

## Figure 5 - Counterfactual Validation (`fig5_counterfactuals.pdf`)

- Layout: 2x2 narrative causal suite.
- Panels:
  - (a) Simulated removal by model x component (mean shift with bootstrap 95% CI).
  - (b) Activation-patching distributions of `delta_final_base` vs `delta_final_patched` with mean-shift arrow.
  - (c) Substitution summary split into two readouts:
    - top: mean `delta_shift` with bootstrap 95% CI,
    - bottom: fraction of pairs with positive `delta_shift`.
  - (d) Span deletion causal test: mean `effect_delta` by span label with bootstrap CI, plus baseline lines from negative controls.
- Data files: `ablation_results.parquet`, `patching_results.parquet`, `span_labels.parquet`, `negative_controls.parquet`.

## Figure 6 - Prompt Journey (`fig6_prompt_journey.pdf`)

- Layout: left structured span panel + right stacked two-panel quantitative view.
- Prompt selection (deterministic): stable-correct prompt prioritized by highest `delta_sign_flips`, then competitor switches and final margin.
- Panels:
  - (a) Structured prompt spans with role fills and evidence/distractor outlines, with deletion-effect labels.
  - (b) `delta(l)` vs real layer index with tail-window overlay (`tau=8`), competitor strip, and sign-flip markers.
  - (c) Component-force traces `s_attn(l)`, `s_mlp(l)`, and `s_attn(l)+s_mlp(l)` vs layer.
- Data files: `decision_metrics.parquet`, `tracing_scalars.parquet`, `span_labels.parquet`, prompt JSONL.

## Figure 7 - Decision Landscape (`fig7_decision_landscape.pdf`)

- Layout: 1x2 full-width panel.

  - (b) Count-masked mean drift field in `(depth, delta)` space.

## Readability and Integrity Gates

The generator enforces layout integrity before writing each PDF:

- no substantial overlapping labels,
- no text/legend/annotation boxes outside figure canvas,
- annotation arrows must use clipping and remain in-axis when data-based.

Any failure raises an explicit error with figure and axis context.
